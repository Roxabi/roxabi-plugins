## Install

### From marketplace

```bash
claude plugin marketplace add Roxabi/roxabi-plugins
claude plugin install roxabi-plugins
```

### From local clone

```bash
git clone https://github.com/Roxabi/roxabi-plugins.git
claude plugin install /path/to/roxabi-plugins
```
