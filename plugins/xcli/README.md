# XCLI Plugin

Claude Code skill for [roxabi-xcli](https://github.com/Roxabi/roxabi-xcli) — Twitter/X from the terminal with cookie auth and write hardening.

## Install

From the Roxabi marketplace:

```bash
claude plugin marketplace add Roxabi/roxabi-plugins
claude plugin install xcli@roxabi-marketplace
```

Or from the dedicated marketplace:

```bash
claude plugin marketplace add Roxabi/roxabi-xcli-marketplace
claude plugin install xcli@roxabi-xcli-marketplace
```
