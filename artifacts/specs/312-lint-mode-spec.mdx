## Expected Behavior

**Dispatch & preload.** `/compress lint <scope>`: mode-exists gate resolves `references/lint.md` (this PR ships it; only `derive` halts after). Phase 0 loads notation.md's `## Core Table` + Grammar + Maintenance (the lazy halves load in lint mode ‚Äî train-B budget rule); glossary absent ‚Üí halt: "lint requires the notation glossary (train B) ‚Äî plugins/shared/references/notation.md not found". **Scope.** Train-A resolution (file|glob|dir|plugin). Vault paths (anything under `get_plugin_data`/`~/.roxabi-vault`) hard-excluded at resolution; excluded paths never appear in the report except one summary line "vault paths excluded: N".

**Step 1 ‚Äî deterministic lint.** The 8 classes per Decision 10's pattern contracts, embedded in lint.md. Hard exclusions applied BEFORE matching: fenced code blocks, **inline code spans** (`` `‚Ä¶` `` ‚Äî the corpus cites glyphs in inline code everywhere; without this the mode lints the glossary's own deprecation records), frontmatter blocks, spawn-template sections (a heading matching /spawn/i up to the next same-or-higher-level heading, plus any fenced `Task(` block).

**Step 2 ‚Äî report.** `file:line | class | current | proposed` grouped by class with counts; genre profile per file shown (always-on ‚Üí aggressive invariant+pointer advisory; memory/** + MEMORY.md ‚Üí provenance-preserving, never propose removing dates/links; skills/agents/commands ‚Üí full rule set; default ‚Üí skills profile; override via present-choice). Dry-run (default) STOPS here after the class multi-select records intent.

**Step 3 ‚Äî repair (`--fix`).** Mechanical classes: one batch approval ‚Üí Edit with per-file diff preview; semantic: per-file present-choice; `negative-polarity` NEVER auto-applied under any path (queue-only, G1 proposal only when Z exists in source, else `needs external verification`). Batch cap ‚â§10 files; larger scopes chunk. In-flight guard before every write (Decision 7). Every queued repair serializes `{diff_type ‚àà {claude_md, skill, memory_entry, ssot}, target_path, diff_body}` ‚Äî mapping total: always-on‚Üíclaude_md, ssot-style ("user-designated always-on manifests", generic)‚Üíssot, memory‚Üímemory_entry, else‚Üískill.

**Ledger + rules.** One Observation row per run (Decision 8). Operational rules in lint.md: dry-run default, one PR per plugin scope, small batches, cache never edited, no `--force`/`--hard`/`--amend`.
