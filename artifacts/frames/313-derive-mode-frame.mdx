## Problem

The derivation reflex (instances → patterns → principles) has succeeded ≥3 times in this corpus — DP(n)/decision-presentation extraction (~132 call sites), plan-task-schema.md SSoT, the ADR-consolidation principle with its `trigger:`/`ref:` canonical form — **100% manually**. Nothing mechanizes it (issue #313; analysis §2.4):

- compress is structurally blind to cross-file repetition (R1–R10 all intra-file): I/V boilerplate in 15+ dev-core skills, pipeline verify-tables, status-glyph vocabulary go unseen.
- This is the roxabi-cortex PoC headline: derive rehearses the consolidate/Retain **schema** — schema_fit routing, dark matter, conflict handling, provenance, approval queue — on real documents at $0.
- **Verified hazard**: naive R11 extraction is token-NEGATIVE at runtime for cross-skill call sites (pointer forces a ~550-token reference Read where inline cost ~100) — an explicit amortization gate is mandatory; "extraction saves tokens" cannot be assumed.

## Premise Validity

**Success in 6 months:** `/compress derive` produces bounded, honest reports: clusters gated by executable predicates, routed by schema_fit with dark matter respected, every proposal carrying double-entry token arithmetic and full provenance; the report format has been exercised on the dogfood targets; cortex's consolidation-schema decisions reference this PoC's recorded outcomes.

**Failure in 6 months:** By 2027-01: zero derivation ledger rows on real scopes (mode unused), or an emitted proposal shows single-entry token accounting (amortization gate bypassed), or a poor-fit cluster was force-abstracted (dark-matter breach), or any v1 run wrote to a scoped file (report-only breach) — the mechanization was unsound or theater; premise indicted.

**Simplest alternative:** Keep deriving manually (it worked 3 times) and document a checklist for humans.
**Why not simplest:** The manual reflex doesn't scale past what one person notices (15+ I/V instances went unextracted for months); a checklist produces no schema evidence for cortex (routing/conflict/provenance/queue must be exercised, not described); and the amortization hazard is exactly what humans skip — the 3 manual successes never computed the runtime delta that makes half of naive extractions net-negative.
